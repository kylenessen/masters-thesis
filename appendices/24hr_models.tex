\section{24-Hour Robustness Analysis: Candidate Models}

This appendix documents all candidate models tested in the 24-hour robustness analysis. A total of 76 models were evaluated using a fixed 24-hour weather window to validate findings from the sunset window analysis.

\subsection{Model Structure}

All models used Generalized Additive Mixed Models (GAMM) with the following structure:

\begin{itemize}
    \item \textbf{Response variable}: $\sqrt{\Delta\text{Butterflies}_{95th}}$ (square root of change in 95th percentile butterfly count between days)
    \item \textbf{Random effects}: Random intercept for deployment site
    \item \textbf{Correlation structure}: AR(1) autocorrelation for temporal dependence
    \item \textbf{Estimation method}: Restricted Maximum Likelihood (REML)
\end{itemize}

\subsection{Baseline Control Variables}

All models included control terms for:
\begin{itemize}
    \item \texttt{max\_butterflies\_t\_1}: Previous day maximum count (tested as smooth or linear)
    \item \texttt{lag\_duration\_hours}: Weather window duration (24 hours, constant)
\end{itemize}

\subsection{Weather Predictor Variables}

\begin{itemize}
    \item \texttt{temp\_min}: Minimum temperature in 24-hour window ($^\circ$C)
    \item \texttt{temp\_max}: Maximum temperature in 24-hour window ($^\circ$C)
    \item \texttt{temp\_at\_max\_count\_t\_1}: Temperature when max count occurred ($^\circ$C)
    \item \texttt{wind\_max\_gust}: Maximum wind gust in 24-hour window (m/s)
    \item \texttt{sum\_butterflies\_direct\_sun}: Cumulative butterflies in direct sun
    \item \texttt{s()}: Thin-plate regression spline smooth term
    \item \texttt{ti()}: Tensor product interaction smooth
\end{itemize}

\subsection{Model Selection Results}

Model M31 (Interaction: wind\_max\_gust $\times$ sum\_butterflies\_direct\_sun with smooth baseline) was identified as the best-fit model based on AICc. Of the 76 candidate models, 72 converged successfully. This analysis validated the key finding from the sunset window analysis: the interaction between wind and direct sun exposure is the strongest predictor of roost abandonment. Complete model selection statistics are reported in the main text.

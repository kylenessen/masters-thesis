\section{Threshold Wind Disruption Analysis: Candidate Models}

This appendix documents all candidate models tested in the threshold wind disruption sensitivity analysis. A total of 52 models were evaluated using a binary threshold-based wind metric to compare with the continuous maximum gust approach.

\subsection{Model Structure}

All models used Generalized Additive Mixed Models (GAMM) with the following structure:

\begin{itemize}
    \item \textbf{Response variable}: $\sqrt[3]{\Delta\text{Butterflies}}$ (cube root of butterfly count change between consecutive 30-minute intervals)
    \item \textbf{Random effects}: Random intercept for deployment site
    \item \textbf{Correlation structure}: AR(1) autocorrelation within observation days
    \item \textbf{Estimation method}: Restricted Maximum Likelihood (REML)
\end{itemize}

\subsection{Variable Definitions}

\begin{itemize}
    \item \texttt{total\_butterflies\_t\_lag}: Previous butterfly count (lagged)
    \item \texttt{minutes\_above\_threshold}: Minutes with wind speed $\geq$ 2 m/s
    \item \texttt{temperature\_avg}: Average temperature ($^\circ$C)
    \item \texttt{butterflies\_direct\_sun\_t\_lag}: Butterflies in direct sun (lagged)
    \item \texttt{time\_within\_day\_t}: Time since sunrise (hours)
    \item \texttt{s()}: Thin-plate regression spline smooth term
    \item \texttt{ti()}: Tensor product interaction smooth
    \item \texttt{*}: Linear interaction term
\end{itemize}

\subsection{Model Selection Results}

Model T50 was identified as the best-fit model based on Akaike Information Criterion (AIC). This model structure mirrors M50 from the continuous wind analysis, enabling direct comparison between threshold-based and continuous wind metrics. Complete model selection statistics are reported in the main text.
